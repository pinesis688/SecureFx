# SecureFx 加密工具箱

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Security](https://img.shields.io/badge/Security-Client%20Side-green.svg)](https://github.com)
[![Web Crypto API](https://img.shields.io/badge/API-Web%20Crypto-blue.svg)](https://developer.mozilla.org/en-US/docs/Web/API/Web_Crypto_API)

一款功能完善的浏览器端加密工具箱，所有加密操作均在本地完成，数据不经过任何服务器。支持 AES-GCM/CBC 对称加密、RSA/ECC 非对称加密、ECDSA 数字签名、SHA-2/SHA-3 哈希计算、商用密码随机性检测、密码强度检测等功能。

## 功能特性

### 核心加密功能

| 功能模块 | 描述 | 算法支持 |
|---------|------|----------|
| **文件加密** | 大文件分块加密，支持进度显示 | AES-256-GCM / AES-256-CBC + HMAC |
| **文本加密** | 文本字符串加密 | AES-256-GCM / AES-256-CBC |
| **RSA 加密** | 非对称加密，适合小数据 | RSA-OAEP (2048/4096 bit) |
| **ECC 加密** | 椭圆曲线加密，效率更高 | ECDSA P-256 + AES-GCM (ECIES) |

### 数字签名

| 功能 | 描述 |
|------|------|
| **文本签名** | 对文本内容进行签名验证 |
| **文件签名** | 生成独立的 `.sig` 签名文件 |
| **签名+加密** | 加密同时签名，一站式操作 |
| **篡改检测** | 自动检测数据是否被篡改 |

### 密钥派生函数 (KDF)

| 算法 | 特点 | 推荐场景 |
|------|------|----------|
| **Argon2id** | 抗 GPU/ASIC 攻击，内存硬函数 | 高安全需求（推荐） |
| **Scrypt** | 内存硬函数，广泛支持 | 兼容性要求 |

### 哈希算法

| 算法 | 输出长度 | 说明 |
|------|----------|------|
| SHA-256 | 32 字节 | 推荐使用 |
| SHA-384 | 48 字节 | 高安全需求 |
| SHA-512 | 64 字节 | 高安全需求 |
| SHA3-256 | 32 字节 | NIST 新标准 |
| SHA3-512 | 64 字节 | NIST 新标准 |
| SM3 | 32 字节 | 国密算法 |
| MD5 | 16 字节 | 不推荐，仅用于兼容 |

### 密码强度检测器

专业的密码安全评估工具，提供：

- **实时强度评估** - 5级强度评分（非常弱/弱/一般/强/非常强）
- **破解时间预估** - 离线暴力攻击场景下的破解时间
- **密码熵值计算** - 信息熵量化评估
- **字符类型检查** - 大小写字母、数字、符号检测
- **安全警告** - 重复字符、连续序列检测
- **优化建议** - 智能密码改进建议

### 商用密码随机性检测

基于 NIST SP 800-22 标准的随机性检测工具箱：

| 检测项目 | 说明 |
|----------|------|
| 单比特频数检测 | 检测序列中0和1的比例是否接近1:1 |
| 块内频数检测 | 检测每个块内比特分布的均匀性 |
| 扑克检测 | 检测m位模式的分布是否均匀 |
| 游程总数检测 | 检测连续相同比特的游程总数 |
| 块内最大游程检测 | 检测块内最长连续相同比特 |
| 矩阵秩检测 | 检测矩阵秩分布，评估线性依赖性 |
| 离散傅里叶检测 | 检测频域特性，评估周期性模式 |
| 近似熵检测 | 检测序列复杂度 |
| 累加和检测 | 检测累加和分布 |

支持二进制字符串、十六进制数据、文件输入三种方式。

## 安全特性

### 端到端安全

- 所有加密操作在浏览器本地完成
- 数据不经过任何服务器
- 无需网络连接即可使用
- 开源代码，可审计

### 安全审计

内置完整的自检测试，覆盖所有核心功能：

- 错误码系统检测
- 加密模式检测 (AES-GCM/CBC)
- Argon2/Scrypt 密钥派生测试
- ECDSA 签名验证测试
- 文件加密/解密完整性测试
- HMAC 校验测试
- 哈希算法测试
- 随机数生成器测试

### 威胁模型声明

本工具无法防御：
- 恶意软件、键盘记录器
- 系统级攻击、内存dump
- 浏览器恶意扩展

禁止用于：
- 国家秘密
- 金融核心数据
- 等保三级以上数据
- 任何法律禁止的场景
